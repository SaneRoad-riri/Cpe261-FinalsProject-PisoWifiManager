This section outlines the prerequisites and steps required to configure and execute the **PisoWifi Management System** in a local development environment.

### 1. Prerequisites
Ensure the following software is installed on your system before proceeding:
* **Operating System:** Windows 10/11, macOS, or Linux.
* **.NET SDK:** Version 8.0 or higher is required to compile and run the application. [Download .NET 8.0](https://dotnet.microsoft.com/download)
* **Integrated Development Environment (IDE):** Visual Studio 2022 (recommended) or Visual Studio Code.

### 2. Cloning the Repository
To obtain the source code, open your terminal or command prompt and execute the following command:
git clone [https://github.com/SaneRoad-riri/Cpe261-Finals-PisoWifiManager.git](https://github.com/SaneRoad-riri/Cpe261-Finals-PisoWifiManager.git)

Alternatively, you may download the project as a ZIP archive from the repository homepage and extract it to your desired directory.

### 3. Execution Instructions
Method A: Using Visual Studio (Recommended)
1. Navigate to the project directory and open the solution file: FinalsProjectCompact.sln.
2. Allow Visual Studio to restore any missing NuGet packages.
3. Click the Start button in the toolbar or press F5 to compile and launch the application in debug mode.

Method B: Using .NET CLI
1. Open a terminal or command prompt.
2. Navigate to the root directory of the project (where the .csproj file is located).
3. Execute the following command to build and run the application:
   dotnet run
